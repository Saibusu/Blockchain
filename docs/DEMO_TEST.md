**1.** 點選右上角「🦊 連接 MetaMask」按鈕

## 常見問題排查

| 問題 | 原因 | 解法 |
|------|------|------|
| 「Unauthorized: incorrect role」 | MetaMask 帳號未切換正確 | 確認右上角 Account 編號 |
| 「Batch ID already registered」 | 批次 ID 已存在 | 改用不同編號，如 `BATCH-2026-002` |
| 「Batch does not exist or is inactive」 | 批次已召回或 ID 拼錯 | 確認批次編號大小寫一致 |
| 「Production date cannot be in the future」 | 日期選了未來 | 選 2022 或 2023 年的日期 |
| 批次編號沒有自動填入 | 尚未登記或查詢批次 | 先完成 S03，橫幅出現後即自動同步 |
| 合約載入失敗 | 網路或地址錯誤 | 確認 MetaMask 和 Remix 在同一網路 |
